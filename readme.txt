== Description ==

SMS integration for WordPress.

== Installation ==

1. Upload `mobiweb-isms` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter username and password of valid iSMS account(www.isms.com.my) in iSMS Settings page

== Frequently Asked Questions ==

= Where do you get the username and password? =

You can create an iSMS account at the [website](http://isms.com.my/register.php).

= How do you purchase credit? =

Click the Reload Credit link next to the balance amount or visit our [website](http://isms.com.my/buy_reload.php).
